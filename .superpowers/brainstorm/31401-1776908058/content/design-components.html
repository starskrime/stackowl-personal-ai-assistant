The one rule every component follows: takes plain data in, returns an ANSI string out. No timers, no event listeners, no process.stdout calls, no shared mutable state. The renderer holds all state and calls component.render(props) on each frame. This makes every component independently testable — just call render() with props and assert on the output string.
