### Track B — Recommendations (2 items)

**REC-001 → GAP-003 CLOSED: Instinct definition library**

Six instinct definition files created at `owls/noctua/instincts/`:

- `frustration-recovery.md` — keyword triggers on frustration signals; constraint: shift to shorter acknowledgements and confirm understanding before acting
- `deep-focus.md` — triggered by long structured task framing; constraint: minimal interruptions, batch clarifications
- `curiosity-mode.md` — triggered by exploratory or research-intent phrasing; constraint: surface alternatives, ask broadening questions
- `decision-fatigue.md` — triggered by "I don't know / you decide" patterns; constraint: reduce option count, give a clear recommendation
- `celebration.md` — triggered by success language; constraint: acknowledge before moving forward
- `time-pressure.md` — triggered by urgency keywords; constraint: skip preamble, lead with the action

Alongside the file creation, a registry keywords bug was fixed: `loadForOwl()` was silently dropping all parsed keywords before passing them to the fast-path matcher. The bug caused V2 heuristic evaluation to always miss. Fix adds proper keyword propagation and 4-point logging to `loadForOwl()`.

Commits: `7e8ea6d` `af0fd06` `7c2e047`
